# CoreFlow - Sistema de Gestión Comercial Segura

CoreFlow es una aplicación web de gestión comercial desarrollada con Angular, Spring Boot, Spring JPA, MySQL y seguridad JWT. El sistema permite gestionar productos, pedidos, carrito de compras, pagos, comprobantes y operaciones administrativas mediante roles diferenciados para usuarios y administradores.

Este proyecto fue desarrollado como trabajo final del curso, cumpliendo con la implementación de tres procesos de negocio no CRUD utilizando Angular y Spring JPA.
